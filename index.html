gdx++

A cross-platform C++ game engine

Download this project as a .zip file Download this project as a tar.gz file

(ps: By no means we're saying that this library is better than the original. All kudos for Mario and everyone that contributed there. I just found gdx++ analogous to C++ =)

Oh hai

GDX++ is the port of the libgdx java library to C++. Although not complete (missing the OpenGL ES 2 support), the engine is in ready-to-production shape and as a bonus it has:

  • iOS support
  • Full antigrain-improved svg parsing and rendering engine (say goodbye to under or over resolution textures!)

As the original library, gdx++ is released under the Apache license.

To learn how to build and how to deploy under it's different platforms, please check README file.
